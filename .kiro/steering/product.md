# 产品定义

## 核心价值

为AI时代独立创业者(OPC)提供**政策查询**、**技能交易**、**社区成长**的一站式服务平台，同时支持AI Agent通过API接入。

## 核心功能

1. **政策引擎**: 上海徐汇龙华OPC政策聚合、智能匹配、AI友好的结构化API
2. **能力市场**: OPC接单派单平台，服务需求双向匹配
3. **创业者社区**: 问答交流、经验分享
4. **成长学院**: 培训课程、导师对接

## 用户角色

- **创业者**: 查政策、接订单、学技能
- **需求方**: 发布需求、找服务商
- **导师**: 提供培训咨询
- **AI Agent**: 通过API为用户代劳

## 关键指标

- 政策覆盖: 100% 龙华街道/徐汇区OPC相关政策
- 响应时间: API < 200ms
- 并发支持: 1万+

## 差异化优势

1. AI-First架构: 支持MCP协议，AI Agent原生接入
2. 本地深耕: 专注龙华街道，政策精准
3. 交易闭环: 从"知道政策"到"接到订单"
